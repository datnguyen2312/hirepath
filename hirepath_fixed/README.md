# HirePath Site Focused MVP

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./start.sh
```

Then open `http://127.0.0.1:8000`.

## Codespaces

Use the same commands but open the forwarded port 8000 from the PORTS tab.
If you see a stale 502 page, stop forwarding port 8000 and re-add it.
